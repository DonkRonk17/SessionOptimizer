# SessionOptimizer Branding Prompts

Visual branding prompts for DALL-E generation following Beacon HQ Visual System v1.

---

## Prompt 1: Title Card (16:9 Landscape)

```
Create a professional technology title card for "SessionOptimizer" - an AI session efficiency analyzer tool. 

Design elements:
- Deep navy blue background (#0a0e27) with subtle digital grid pattern
- Large glowing efficiency gauge/meter as the central element, showing optimization progress
- Flowing data streams connecting session nodes in a network visualization
- The gauge needle pointing to "OPTIMIZED" zone in bright cyan (#00d4ff)
- Text "SessionOptimizer" in clean, modern sans-serif font
- Subtitle "AI Session Efficiency Analyzer" below in lighter weight
- Small "Q-Mode Tool #18" badge in corner
- Accent elements: charts, token counters, efficiency percentages floating
- Professional tech aesthetic with subtle glow effects
- 16:9 aspect ratio, suitable for GitHub README header
```

---

## Prompt 2: Logo Mark (1:1 Square)

```
Design a minimalist logo mark for "SessionOptimizer" - an AI efficiency analysis tool.

Design elements:
- Square format (1:1 ratio)
- Dark navy background (#0a0e27)
- Central icon: stylized efficiency gauge or speedometer
- The gauge shows optimization with cyan (#00d4ff) fill
- Small data points or nodes around the gauge representing sessions
- Clean geometric design
- Works at small sizes (favicon, app icon)
- Professional, tech-forward aesthetic
- Subtle glow effect on the cyan elements
```

---

## Prompt 3: Horizontal Logo (3:1 Wide)

```
Create a horizontal logo for "SessionOptimizer" combining icon and wordmark.

Design elements:
- Wide format (3:1 ratio)
- Dark navy background (#0a0e27) or transparent
- Left side: efficiency gauge icon with cyan fill
- Right side: "SessionOptimizer" text in modern sans-serif
- Text in white or cyan gradient
- Small efficiency percentage badge (e.g., "99%") near icon
- Professional, clean design suitable for headers
- Beacon HQ style: navy, cyan accents, tech aesthetic
```

---

## Prompt 4: App Icon (1:1 Simplified)

```
Design a simplified app icon for "SessionOptimizer".

Design elements:
- Square format with rounded corners (iOS/Android style)
- Dark navy background (#0a0e27)
- Ultra-simplified efficiency gauge icon
- Single cyan (#00d4ff) accent color
- Bold, recognizable at 64x64 pixels
- Clean lines, no text
- Subtle gradient or glow effect
- Works on both light and dark backgrounds
```

## Color Palette

- **Primary Background:** #0a0e27 (Deep Navy)
- **Primary Accent:** #00d4ff (Bright Cyan)
- **Secondary Accent:** #00ff88 (Success Green)
- **Warning Accent:** #ffaa00 (Orange)
- **Error Accent:** #ff4444 (Red)
- **Text Primary:** #ffffff (White)
- **Text Secondary:** #a0a0a0 (Gray)

- **Theme:** Efficiency, optimization, measurement
